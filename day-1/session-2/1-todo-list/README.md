# Install using:

## Install Node

```
# Download and install fnm:
winget install Schniz.fnm

# Download and install Node.js:
fnm install 22

# Verify the Node.js version:
node -v # Should print "v22.12.0".

# Verify npm version:
npm -v # Should print "10.9.0".

```

## Repo Installation
```
npm create vite@latest
# follow the manual
cd <app-name>
npm install
npm run dev

```
